# テストデータ

このディレクトリには、アプリケーションのテストに使用するサンプルCSVファイルが格納されています。

## 📁 ファイル一覧

### 算定基礎_介護.csv
- **説明**: 介護保険サービスの算定データ
- **形式**: Shift_JIS, CSV
- **レコード数**: 約420行
- **利用者数**: 29名
- **出力元**: 介護ソフト（CAREKARTE）の「算定一覧」

### 算定基礎_予防.csv
- **説明**: 介護予防サービスの算定データ
- **形式**: Shift_JIS, CSV
- **出力元**: 介護ソフト（CAREKARTE）の「算定一覧」
- **ステータス**: 未実装（Phase 2で対応予定）

---

## 🔒 個人情報保護

### サンプルデータの取り扱い
- ✅ **氏名**: 伏字化済み（実在の人物ではありません）
- ✅ **個人情報**: 本人を特定できる情報は含まれていません
- ✅ **利用目的**: 開発・テスト専用

### データ構造
本番のCSVファイルと同じ形式ですが、個人情報は以下のように処理されています：
- 氏名: 架空の名前に置き換え
- その他の個人識別情報: すべて削除または匿名化

---

## 📊 CSVファイル形式

### 列構造（算定基礎_介護.csv）
| 列番号 | 列名 | 説明 |
|--------|------|------|
| 0 | 対象フラグ | 'False'固定 |
| 1 | 利用者名 | 氏名（先頭行のみ） |
| 2 | 時間 | サービス提供時間 |
| 3 | 基本 | '基本'固定 |
| 4 | サービスコード | 06xxxx/07xxxx/08xxxx |
| 5-11 | 月〜日 | 週間パターン（1/-） |
| 12-15 | その他 | （未使用） |

### サービスコード分類
- **06xxxx**: 訪問介護等
- **07xxxx**: 通所介護等（通い）
- **08xxxx**: 短期入所等（宿泊）

---

## 🎯 使用方法

### 1. アプリでの取り込み
1. ブラウザでアプリを開く（`index.html`）
2. ツールバーの「算定一覧を取り込む」ボタンをクリック
3. `算定基礎_介護.csv`を選択
4. 自動的に29名分の週間パターンが取り込まれます

### 2. 期待される動作
- 利用者マスタに29名が登録される
- 週間パターンが月間予定に自動展開される
- 通所・訪問・宿泊のすべてが表示される

---

## 📝 テストシナリオ

### 基本機能テスト
1. **CSV取り込み**
   - ファイルを選択して取り込み
   - 29名すべてが表示されることを確認

2. **データ表示**
   - 通所（○）、訪問（数字）、宿泊（入/退）が正しく表示される
   - 日付列が上下で揃っている（スクロール同期）

3. **編集機能**（Phase 2）
   - セルをクリックして編集
   - 自動保存される

4. **定員チェック**
   - 通所15名、宿泊9名の定員超過を検出
   - 警告が表示される

## 🔄 データ更新

### テストデータの更新手順
1. CAREKARTEから最新の算定一覧をエクスポート
2. 個人情報を匿名化
3. このディレクトリに上書き保存
4. Gitにコミット

### 注意事項
- ⚠️ 本番データを直接コミットしない
- ⚠️ 個人情報が含まれていないか必ず確認
- ✅ ファイルサイズは数百KB以内に収める
