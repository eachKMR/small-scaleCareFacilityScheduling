# Phase 1-B テスト仕様書 v2.1

**作成日**: 2025年11月18日  
**対象**: セル編集機能（CellEditor）  
**バージョン**: 2.1（ステータスバー表示対応）

---

## 📋 v2.1の変更点

### 変更内容
- ❌ トースト表示（画面上部中央）
- ✅ ステータスバー表示（画面下部固定）

### 変更理由
1. **予定表を一切隠さない** ← 最大のメリット
2. **位置が固定** - ユーザーの視線が定まる
3. **VS Code的UI** - 開発者に馴染みのあるUI
4. **実装がシンプル** - z-indexの管理不要

---

## 📋 テスト目的

Phase 1-Bで実装した2ステップクリック方式のセル編集機能が要件定義書通りに動作することを確認する。

**対象機能**:
- セルのクリック操作
- 2ステップクリック方式による宿泊期間作成
- 右クリックメニュー
- ホバーツールチップ
- フォーカス管理
- ステータスバー表示 ★v2.1で変更

---

## 🖥️ テスト環境

### 必要な環境
- ブラウザ: Chrome / Edge / Firefox（最新版）
- ローカルサーバー起動済み
- CSVデータ取り込み済み（利用者あり）

### 起動方法
```bash
# HTTPサーバー起動

http://localhost:8000
```
