# 小規模多機能利用調整システム

**プロジェクト開始日**: 2025年11月13日  
**現在のステータス**: Phase 1-A実装中（CSVService）  
**最終更新**: 2025年11月15日

---

## 📖 プロジェクト概要

小規模多機能型居宅介護の利用予定を効率的に調整するためのWebアプリケーション。ケアカルテから出力されたCSVを取り込み、29名分の月間予定（通い・泊り・訪問）を視覚的に管理し、定員チェックを行いながら調整できます。

---

## 🎯 プロジェクトの目的

1. **算定基礎の効率的な取り込み**
   - ケアカルテCSV → 週間パターン抽出 → 月間自動展開

2. **視覚的な定員管理**
   - 58行×33列のグリッド表示
   - 前半・後半の分離カウント
   - リアルタイムな定員状況表示

3. **柔軟な調整機能**
   - ドラッグ&ドロップでの期間設定
   - 右クリックメニューでの細かい調整
   - 削除⇔復元機能

4. **Excel連携**
   - 調整結果のExcel出力
   - ケアカルテへの転記を効率化
