# データモデル設計書 v3.0

**作成日**: 2025年11月13日 
**最終更新**: 2025年11月17日 
**対象**: Phase 0（ゼロベース） → Phase 1 へのデータモデル  
**目的**: 特別セル対応、33セル構造、データ保存戦略の確定

---

## 📊 変更概要（v2.0 → v3.0）

### 主な変更点
1. **特別セル**：月またぎ宿泊対応（33セル構造）
2. **データ保存キー**：`prevMonth`, `nextMonth`の確定
3. **定員カウント**：退所日は泊りカウントなし
4. **データ保持戦略**：1ヶ月のみ、自動削除
5. **利用者マスタ**：ソートID追加、DEFAULT_USERS廃止
6. **セルタイプ統合**: am/pm → dayStayに統合（v3.1で表記修正）

#### After（v3.0）

```javascript
class ScheduleCell {
  constructor(userId, date, cellType) {
    this.userId = userId;
    this.date = date;                // 通常セル: "2025-12-05"
                                     // 特別セル: "2025-12-prevMonth" | "2025-12-nextMonth"
    this.cellType = cellType;        // "dayStay" | "visit"
    this.inputValue = "";            // "○", "◓", "◒", "入", "退", 数値
                                     // 特別セル: "12/28入", "1/3退"
    this.note = "";
    this.actualFlags = {
      day: false,
      stay: false,
      visit: 0,
      halfDayType: 'full'           // 'full' | 'morning' | 'afternoon'
    };
    
    // 削除⇔復元機能
    this.deletedValue = null;
    this.deletedAt = null;
  }
  
  // 既存メソッド（記号対応）
  isEmpty() {
    return this.inputValue === "" && this.deletedValue === null;
  }
  
  isStayStart() {
    return this.inputValue === "入";
  }
  
  isStayEnd() {
    return this.inputValue === "退";
  }
  
  // 🆕 特別セル判定メソッド
  isPrevMonthCell() {
    return this.date.includes('prevMonth');
  }
  
  isNextMonthCell() {
    return this.date.includes('nextMonth');
  }
  
  isSpecialCell() {
    return this.isPrevMonthCell() || this.isNextMonthCell();
  }
  
  // 🆕 記号判定メソッド
  isFullDay() {
    return this.inputValue === AppConfig.SYMBOLS.FULL_DAY;  // "○"
  }
  
  isMorning() {
    return this.inputValue === AppConfig.SYMBOLS.MORNING;  // "◓"
  }
  
  isAfternoon() {
    return this.inputValue === AppConfig.SYMBOLS.AFTERNOON;  // "◒"
  }
  
  // 削除機能
  delete() {
    if (this.inputValue !== "") {
      this.deletedValue = this.inputValue;
      this.deletedAt = Date.now();
      this.inputValue = "";
      Logger?.debug(`Cell deleted: ${this.userId} ${this.date}`);
    }
  }
  
  // 復元機能
  restore() {
    if (this.canRestore()) {
      this.inputValue = this.deletedValue;
      this.deletedValue = null;
      this.deletedAt = null;
      Logger?.info(`Cell restored: ${this.userId} ${this.date}`);
      return true;
    }
    return false;
  }
  
  // 復元可能判定
  canRestore() {
    if (this.deletedValue === null) return false;
    const elapsed = Date.now() - this.deletedAt;
    return elapsed < AppConfig.RESTORE_TIMEOUT;
  }
  
  // 削除状態のクリア（タイムアウト時）
  clearDeletedState() {
    this.deletedValue = null;
    this.deletedAt = null;
  }
  
  // 🆕 定員カウント用（修正版）
  getDayCountContribution() {
    // 特別セルは当月の定員カウントに含めない
    if (this.isSpecialCell()) {
      return { morning: 0, afternoon: 0 };
    }
    
    if (this.isFullDay() || this.isStayStart()) {
      return { morning: 1, afternoon: 1 };
    } else if (this.isStayEnd()) {
      // ★重要：退所日は泊りカウントしない
      // 通いとしてはカウントするが、actualFlags.stayはfalse
      return { morning: 1, afternoon: 1 };
    } else if (this.isMorning()) {
      return { morning: 1, afternoon: 0 };
    } else if (this.isAfternoon()) {
      return { morning: 0, afternoon: 1 };
    } else if (this.actualFlags.stay) {
      // 宿泊中（○）
      return { morning: 1, afternoon: 1 };
    }
    return { morning: 0, afternoon: 0 };
  }
  
  toJSON() {
    return {
      userId: this.userId,
      date: this.date,
      cellType: this.cellType,
      inputValue: this.inputValue,
      note: this.note,
      actualFlags: this.actualFlags,
      deletedValue: this.deletedValue,
      deletedAt: this.deletedAt
    };
  }
  
  static fromJSON(data) {
    const cell = new ScheduleCell(data.userId, data.date, data.cellType);
    cell.inputValue = data.inputValue || "";
    cell.note = data.note || "";
    cell.actualFlags = data.actualFlags || { 
      day: false, 
      stay: false, 
      visit: 0, 
      halfDayType: 'full' 
    };
    cell.deletedValue = data.deletedValue || null;
    cell.deletedAt = data.deletedAt || null;
    return cell;
  }
}
```

**変更点**:
- ✅ 特別セル判定メソッド追加
- ✅ `getDayCountContribution()`で特別セルを除外
- ✅ 退所日の扱いを明確化

```javascript
class StayPeriod {

    this.endDate = endDate;         // 退所日
    this.userId = userId;
  }
}
```

#### After（v3.0）

```javascript
class StayPeriod {
  constructor(startDate, endDate, userId) {
    this.startDate = startDate;     // 入所日（実日付）
    this.endDate = endDate;         // 退所日（実日付、月をまたぐ可能性あり）
    this.userId = userId;
  }
  
  // 期間内全日付を取得
  getDates() {
    const dates = [];
    const start = new Date(this.startDate);
    const end = new Date(this.endDate);
    
    for (let d = new Date(start); d <= end; d.setDate(d.getDate() + 1)) {
      dates.push(DateUtils.formatDate(d));
    }
    
    return dates;
  }
  
  // 日付が期間内か判定
  contains(date) {
    const d = new Date(date);
    const start = new Date(this.startDate);
    const end = new Date(this.endDate);
    return d >= start && d <= end;
  }
  
  // 宿泊日数（退所日は含まない）
  getDuration() {
    const start = new Date(this.startDate);
    const end = new Date(this.endDate);
    const diffTime = Math.abs(end - start);
    const diffDays = Math.ceil(diffTime / (1000 * 60 * 60 * 24));
    return diffDays; // 退所日は泊りカウントしないため
  }
  
  // 🆕 当月にかかる日付のみを取得
  getDatesInMonth(yearMonth) {
    const allDates = this.getDates();
    return allDates.filter(date => date.startsWith(yearMonth));
  }
  
  toJSON() {
    return {
      startDate: this.startDate,
      endDate: this.endDate,
      userId: this.userId
    };
  }
  
  static fromJSON(data) {
    return new StayPeriod(data.startDate, data.endDate, data.userId);
  }
}
```

**変更点**:
- ✅ `getDatesInMonth()`追加（当月分のみ取得）
- ✅ `getDuration()`のコメント更新（退所日は泊りカウントしない）

### 4. ScheduleCalendar クラス

#### Before（v2.0）

```javascript
class ScheduleCalendar {
  constructor(userId, yearMonth) {
    this.userId = userId;
    this.yearMonth = yearMonth;     // "2025-12"
    this.cells = new Map();         // Map<date, ScheduleCell>
    this.stayPeriods = [];          // StayPeriod[]
  }
}
```

#### After（v3.0）

```javascript
class ScheduleCalendar {
  constructor(userId, yearMonth) {
    this.userId = userId;
    this.yearMonth = yearMonth;     // "2025-12"
    this.cells = new Map();         // Map<date, ScheduleCell>
                                    // 通常セル: "2025-12-05_dayStay"
                                    // 特別セル: "2025-12-prevMonth_dayStay", "2025-12-nextMonth_dayStay"
    this.stayPeriods = [];          // StayPeriod[]（実日付ベース）
  }
  
  // セルを設定
  setCell(date, cellType, value) {
    const key = `${date}_${cellType}`;
    
    if (!this.cells.has(key)) {
      this.cells.set(key, new ScheduleCell(this.userId, date, cellType));
    }
    
    const cell = this.cells.get(key);
    cell.inputValue = value;
  }
  
  // セルを取得
  getCell(date, cellType) {
    const key = `${date}_${cellType}`;
    return this.cells.get(key) || null;
  }
  
  // 🆕 特別セルを取得
  getPrevMonthCell(cellType = 'dayStay') {
    const key = `${this.yearMonth}-prevMonth_${cellType}`;
    return this.cells.get(key) || null;
  }
  
  getNextMonthCell(cellType = 'dayStay') {
    const key = `${this.yearMonth}-nextMonth_${cellType}`;
    return this.cells.get(key) || null;
  }
  
  // 🆕 特別セルを設定
  setPrevMonthCell(value, cellType = 'dayStay') {
    const date = `${this.yearMonth}-prevMonth`;
    this.setCell(date, cellType, value);
  }
  
  setNextMonthCell(value, cellType = 'dayStay') {
    const date = `${this.yearMonth}-nextMonth`;
    this.setCell(date, cellType, value);
  }
  
  // 入所〜退所を計算
  calculateStayPeriods() {
    this.stayPeriods = [];
    
    const dayStayCells = [];
    
    // 通常セルのみを取得（特別セルは除外）
    for (const [key, cell] of this.cells) {
      if (cell.cellType === 'dayStay' && !cell.isSpecialCell()) {
        dayStayCells.push(cell);
      }
    }
    
    // 日付順にソート
    dayStayCells.sort((a, b) => a.date.localeCompare(b.date));
    
    let stayStart = null;
    
    for (const cell of dayStayCells) {
      if (cell.isStayStart()) {
        stayStart = cell.date;
      } else if (cell.isStayEnd() && stayStart !== null) {
        this.stayPeriods.push(new StayPeriod(stayStart, cell.date, this.userId));
        stayStart = null;
      }
    }
    
    // 🆕 月をまたぐ宿泊を処理
    this.calculateCrossMonthStay();
  }
  
  // 🆕 月をまたぐ宿泊を処理
  calculateCrossMonthStay() {
    const prevMonthCell = this.getPrevMonthCell();
    const nextMonthCell = this.getNextMonthCell();
    
    // 前月から継続する宿泊
    if (prevMonthCell && prevMonthCell.inputValue) {
      // "12/28入" → startDate: "2024-12-28"
      const prevMonth = DateUtils.getPreviousMonth(this.yearMonth);
      const day = parseInt(prevMonthCell.inputValue.split('/')[1]);
      const startDate = `${prevMonth}-${String(day).padStart(2, '0')}`;
      
      // 当月内の退所を探す
      const dayStayCells = [];
      for (const [key, cell] of this.cells) {
        if (cell.cellType === 'dayStay' && !cell.isSpecialCell()) {
          dayStayCells.push(cell);
        }
      }
      dayStayCells.sort((a, b) => a.date.localeCompare(b.date));
      
      const firstCheckOut = dayStayCells.find(cell => cell.isStayEnd());
      if (firstCheckOut) {
        this.stayPeriods.push(new StayPeriod(startDate, firstCheckOut.date, this.userId));
      }
    }
    
    // 翌月まで継続する宿泊
    if (nextMonthCell && nextMonthCell.inputValue) {
      // "1/3退" → endDate: "2026-01-03"
      const nextMonth = DateUtils.getNextMonth(this.yearMonth);
      const day = parseInt(nextMonthCell.inputValue.split('/')[1]);
      const endDate = `${nextMonth}-${String(day).padStart(2, '0')}`;
      
      // 当月内の入所を探す
      const dayStayCells = [];
      for (const [key, cell] of this.cells) {
        if (cell.cellType === 'dayStay' && !cell.isSpecialCell()) {
          dayStayCells.push(cell);
        }
      }
      dayStayCells.sort((a, b) => b.date.localeCompare(a.date)); // 降順
      
      const lastCheckIn = dayStayCells.find(cell => cell.isStayStart());
      if (lastCheckIn) {
        this.stayPeriods.push(new StayPeriod(lastCheckIn.date, endDate, this.userId));
      }
    }
  }
  
  // stay/dayフラグを計算
  calculateAllFlags() {
    // 全セルのフラグをリセット
    for (const cell of this.cells.values()) {
      cell.actualFlags.day = false;
      cell.actualFlags.stay = false;
      cell.actualFlags.visit = 0;
    }
    
    // 宿泊期間に基づいてフラグを設定
    for (const period of this.stayPeriods) {
      const dates = period.getDatesInMonth(this.yearMonth); // 🆕 当月分のみ
      
      for (const date of dates) {
        const cell = this.getCell(date, 'dayStay');
        if (cell) {
          cell.actualFlags.day = true;
          
          // ★重要：退所日は泊りフラグを立てない
          if (date !== period.endDate) {
            cell.actualFlags.stay = true;
          }
        }
      }
    }
    
    // 通いセルのフラグを設定
    for (const cell of this.cells.values()) {
      if (cell.cellType === 'dayStay' && !cell.isSpecialCell()) {
        if (cell.isFullDay() || cell.isMorning() || cell.isAfternoon()) {
          cell.actualFlags.day = true;
        }
      }
    }
    
    // 訪問セルのフラグを設定
    for (const cell of this.cells.values()) {
      if (cell.cellType === 'visit' && !cell.isSpecialCell()) {
        const visitCount = parseInt(cell.inputValue) || 0;
        cell.actualFlags.visit = visitCount;
      }
    }
  }
  
  // 月内の全日付を取得（33セル構造）
  getDaysInMonth() {
    const dates = DateUtils.getDatesInMonth(this.yearMonth);
    
    // 特別セルを追加
    return [
      `${this.yearMonth}-prevMonth`,
      ...dates,
      `${this.yearMonth}-nextMonth`
    ];
  }
  
  toJSON() {
    const cellsArray = [];
    for (const [key, cell] of this.cells) {
      cellsArray.push({
        key: key,
        ...cell.toJSON()
      });
    }
    
    return {
      userId: this.userId,
      yearMonth: this.yearMonth,
      cells: cellsArray,
      stayPeriods: this.stayPeriods.map(p => p.toJSON())
    };
  }
  
  static fromJSON(data) {
    const calendar = new ScheduleCalendar(data.userId, data.yearMonth);
    
    // セルを復元
    if (data.cells) {
      for (const cellData of data.cells) {
        const cell = ScheduleCell.fromJSON(cellData);
        calendar.cells.set(cellData.key, cell);
      }
    }
    
    // 宿泊期間を復元
    if (data.stayPeriods) {
      calendar.stayPeriods = data.stayPeriods.map(p => StayPeriod.fromJSON(p));
    }
    
    return calendar;
  }
}
```

**変更点**:
- ✅ 特別セルのgetter/setter追加
- ✅ `calculateCrossMonthStay()`追加
- ✅ `getDaysInMonth()`を33セル構造に対応
- ✅ `calculateAllFlags()`で退所日の泊りフラグを立てない

#### After（v3.0）

```javascript
class DailyCapacity {
  constructor(date, dayLimit = 15, stayLimit = 9) {
    this.date = date;
    
    // 🆕 前半・後半の分離カウント
    this.dayCountMorning = 0;      // 前半の通い利用者数
    this.dayCountAfternoon = 0;    // 後半の通い利用者数
    this.dayCount = 0;             // 全日の通い利用者数（互換性のため残す）
    
    this.stayCount = 0;            // 泊り利用者数
    this.visitCount = 0;           // 訪問回数合計
    this.dayLimit = dayLimit;
    this.stayLimit = stayLimit;
  }
  
  // 🆕 前半後半の最大値を取得
  getMaxDayCount() {
    return Math.max(this.dayCountMorning, this.dayCountAfternoon);
  }
  
  // 修正：前半後半を考慮
  isOverCapacity() {
    const maxDayCount = this.getMaxDayCount();
    return maxDayCount > this.dayLimit || this.stayCount > this.stayLimit;
  }
  
  // 修正：前半後半を考慮
  getDayOverflow() {
    const maxDayCount = this.getMaxDayCount();
    return Math.max(0, maxDayCount - this.dayLimit);
  }
  
  // 🆕 前半のオーバーフロー
  getMorningOverflow() {
    return Math.max(0, this.dayCountMorning - this.dayLimit);
  }
  
  // 🆕 後半のオーバーフロー
  getAfternoonOverflow() {
    return Math.max(0, this.dayCountAfternoon - this.dayLimit);
  }
  
  getStayOverflow() {
    return Math.max(0, this.stayCount - this.stayLimit);
  }
  
  // 修正：前半後半の利用率を追加
  getUtilizationRate() {
    return {
      day: (this.dayCount / this.dayLimit) * 100,
      dayMorning: (this.dayCountMorning / this.dayLimit) * 100,    // 🆕
      dayAfternoon: (this.dayCountAfternoon / this.dayLimit) * 100, // 🆕
      stay: (this.stayCount / this.stayLimit) * 100
    };
  }
  
  // 🆕 定員状況の記号を取得
  getCapacitySymbol() {
    const maxDayCount = this.getMaxDayCount();
    const rate = (maxDayCount / this.dayLimit) * 100;
    
    if (rate <= 66) return '◎';      // 余裕
    if (rate <= 80) return '○';      // 適正
    if (rate <= 93) return '△';      // 注意
    return '×';                       // 満員
  }
  
  // 🆕 ツールチップ用の詳細データ
  getTooltipData() {
    return {
      full: this.dayCount,
      morning: this.dayCountMorning,
      afternoon: this.dayCountAfternoon,
      morningOverflow: this.getMorningOverflow(),
      afternoonOverflow: this.getAfternoonOverflow()
    };
  }
  
  toJSON() {
    return {
      date: this.date,
      dayCount: this.dayCount,
      dayCountMorning: this.dayCountMorning,
      dayCountAfternoon: this.dayCountAfternoon,
      stayCount: this.stayCount,
      visitCount: this.visitCount,
      dayLimit: this.dayLimit,
      stayLimit: this.stayLimit
    };
  }
}
```

**変更点**:
- ✅ 前半後半カウントの追加
- ✅ `getMaxDayCount()`追加

### 6. ServiceCapacity クラス

#### Before（v2.0）

```javascript
class ServiceCapacity {
  constructor(calendars) {
    this.calendars = calendars;  // ScheduleCalendar[]
  }
  
  checkDate(date) {
    const capacity = new DailyCapacity(date);
    
    this.calendars.forEach(calendar => {
      const cell = calendar.getCell(date, 'dayStay');
      if (cell && cell.actualFlags.day) {
        capacity.dayCount++;
      }
      if (cell && cell.actualFlags.stay) {
        capacity.stayCount++;
      }
      
      const visitCell = calendar.getCell(date, 'visit');
      if (visitCell) {
        capacity.visitCount += visitCell.actualFlags.visit;
      }
    });
    
    return capacity;
  }
}
```

#### After（v3.0）

```javascript
class ServiceCapacity {
  constructor(calendars) {
    this.calendars = calendars;  // ScheduleCalendar[]
  }
  
  // 修正：前半後半カウント追加
  checkDate(date) {
    const capacity = new DailyCapacity(date);
    
    this.calendars.forEach(calendar => {
      const cell = calendar.getCell(date, 'dayStay');
      
      if (cell && cell.actualFlags.day) {
        // 🆕 前半後半の判定
        const contribution = cell.getDayCountContribution();
        capacity.dayCountMorning += contribution.morning;
        capacity.dayCountAfternoon += contribution.afternoon;
        
        // 互換性のため全日カウントも更新
        if (contribution.morning === 1 && contribution.afternoon === 1) {
          capacity.dayCount++;
        } else {
          capacity.dayCount += 0.5;
        }
      }
      
      if (cell && cell.actualFlags.stay) {
        capacity.stayCount++;
      }
      
      const visitCell = calendar.getCell(date, 'visit');
      if (visitCell) {
        capacity.visitCount += visitCell.actualFlags.visit;
      }
    });
    
    return capacity;
  }
  
  checkMonth(yearMonth) {
    const results = [];
    const dates = DateUtils.getDatesInMonth(yearMonth);
    
    dates.forEach(date => {
      results.push(this.checkDate(date));
    });
    
    return results;
  }
  
  getOverCapacityDates() {
    const yearMonth = this.calendars[0]?.yearMonth;
    if (!yearMonth) return [];
    
    const capacities = this.checkMonth(yearMonth);
    return capacities
      .filter(c => c.isOverCapacity())
      .map(c => c.date);
  }
  
  getSummary() {
    const yearMonth = this.calendars[0]?.yearMonth;
    if (!yearMonth) return null;
    
    const capacities = this.checkMonth(yearMonth);
    
    return {
      totalDays: capacities.length,
      overCapacityDays: capacities.filter(c => c.isOverCapacity()).length,
      averageDayCount: capacities.reduce((sum, c) => sum + c.dayCount, 0) / capacities.length,
      averageDayCountMorning: capacities.reduce((sum, c) => sum + c.dayCountMorning, 0) / capacities.length,    // 🆕
      averageDayCountAfternoon: capacities.reduce((sum, c) => sum + c.dayCountAfternoon, 0) / capacities.length, // 🆕
      maxDayCount: Math.max(...capacities.map(c => c.getMaxDayCount())),
      maxStayCount: Math.max(...capacities.map(c => c.stayCount))
    };
  }
}
```

**変更点**:
- ✅ `checkDate()`で前半後半カウント
- ✅ `getSummary()`で前半後半平均値

## 補足: cellTypeの設計意図

### なぜ`dayStay`に統合したのか

**旧設計（`am`/`pm`分離）の問題点**:
1. 宿泊は終日のため、前半後半の区別が不要
2. 通いの前半後半は、`actualFlags.halfDayType`で管理すれば十分
3. セルタイプが増えると、定員カウントのロジックが複雑化

**新設計（`dayStay`統合）の利点**:
1. セルタイプがシンプル（2種類のみ）
2. 通いと宿泊を同じセルで扱える（UIがシンプル）
3. 前半後半は`actualFlags`で管理（柔軟性）
