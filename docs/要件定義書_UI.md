## 1. 画面レイアウト

### 1.1 全体構成

#### 1.1.1 画面構成

```
┌─────────────────────────────────────────┐
│ツールバー（5%）                          │
│[2025年11月▼] [作業内容を取り込む]       │
│[作業内容を保存] [算定一覧を取り込む] [⚙️]│
├─────────────────────────────────────────┤
│定員状況ヘッダー（10%）- 固定             │
│    │1金│2土│3日│...                    │
│通い│○13│○14│×16│                       │
│泊り│○7 │○8 │△9 │                       │
│訪問│ 5 │ 8 │ 3 │                       │
╞════╪═══╪═══╪═══╡← 太罫線               │
│予定グリッド（85%）- スクロール           │
│ 29名 × 2行 = 58行                       │
│  - 1行目: 通泊行（統合）                │
│  - 2行目: 訪問行                        │
└─────────────────────────────────────────┘
```

#### 1.1.2 画面占有率

| 要素 | 割合 | 備考 |
|------|------|------|
| ツールバー | 5% | 固定 |
| 定員状況ヘッダー | 10% | 固定、横スクロール連動 |
| 予定グリッド | 85% | 縦横スクロール可能 |

**Phase 1-A改善結果**:
- 予定グリッド占有率: 65% → 85%（+20%向上）
- 表示行数: 約10名 → 約20名（2倍）

### 1.4 予定グリッド

#### 1.4.1 構造（33列 × 58行）

**列構成**:
```
列1: 利用者名（固定列）
列2: 前月（左特別セル）
列3-33: 1日〜31日
列34: 翌月（右特別セル）
```

**行構成**:
```
29名 × 2行 = 58行
  - 奇数行（1, 3, 5, ...）: 通泊行
  - 偶数行（2, 4, 6, ...）: 訪問行
```

```css
  cursor: default;
```

### 1.5 スクロール同期（Phase 1-A実装確定）

#### 1.5.1 要件

**課題**:
- 定員状況ヘッダーと予定グリッドは別コンテナ
- 横スクロールが独立していると、日付列がずれて見づらい

**解決策**:
- 定員状況ヘッダーと予定グリッドの**横スクロールを同期**
- どちらをスクロールしても、もう一方も追従する

## 5. UI仕様（レイアウト、操作方法）

### 5.1 操作方法のまとめ ★v1.3.1で更新

**通泊行**:

| セル状態 | クリック | フォーカス | ドラッグ | 右クリック |
|---------|---------|-----------|---------|-----------|
| **空欄** | ○追加 | ✅ | 宿泊期間作成 | メニュー |
| **○** | 空欄に戻す | ✅ | - | メニュー |
| **◓/◒** | 空欄に戻す | ✅ | - | メニュー |
| **入** | フォーカスのみ | ✅ | 入所日調整 | メニュー |
| **退** | フォーカスのみ | ✅ | 退所日調整 | メニュー |
| **◎** | **無反応** | ❌ | - | メニュー（期間クリア） |

**重要な変更点** ★v1.3.1:
1. **空欄クリック** → ○追加
2. **○クリック** → 空欄に戻す（シンプルなトグル）
3. **削除状態なし** → タイマー不要、訪問行と一貫性
4. **空欄ドラッグ** → 宿泊期間作成（入→◎→退）
5. **入・退ドラッグ** → 期間調整（前倒し/延長/短縮）
6. **◎はクリック不可** → 操作不要な中間状態
7. **宿泊期間削除** → 右クリック→クリアのみ

---

**訪問行**:

| 操作 | 動作 |
|------|------|
| クリック | フォーカス + 回数+1（0→1→2→3→0） |
| 右クリック | フォーカス + メニュー表示 |

**詳細は [要件定義書_予定管理_v1.2.1.md](要件定義書_予定管理_v1.2.1.md) を参照**

### 5.3 右クリックメニュー

#### 5.3.1 通泊行のメニュー

**通常セル（空欄、○、◓、◒）**:
```
┌──────────┐
│ 通い全日  │ → ○
│ 前半通い  │ → ◓
│ 後半通い  │ → ◒
│ 入所      │ → 入
│ 退所      │ → 退
│ クリア    │ → 空白
│ --------- │
│ 備考      │ → 備考パネル表示
└──────────┘
```

**宿泊期間中のセル（入、◎、退）**:
```
┌──────────┐
│ クリア    │ → 期間全体を削除
│ --------- │
│ 備考      │ → 備考パネル表示
└──────────┘
```

**設計意図**:
- 宿泊期間は「期間全体」として扱う
- 個別の記号変更は不可（混乱を防ぐ）
- 期間調整は「ドラッグ」で行う

```
┌──────────┐

│ 2回       │ → 2
│ 3回       │ → 3
│ 直接入力  │ → テキスト入力ダイアログ
│ クリア    │ → 空白
│ --------- │
│ 備考      │ → 備考パネル表示
└──────────┘
```
