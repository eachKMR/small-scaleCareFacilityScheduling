## 1. 画面レイアウト

### 1.1 全体構成

```
┌─────────────────────────────────────┐
│ ツールバー                           │← 固定
├─────────────────────────────────────┤
│ 定員状況ヘッダー                    │← 固定
├─────────────────────────────────────┤
│                                     │
│ 予定グリッド（スクロール可能）      │← メイン
│                                     │
│                                     │
├─────────────────────────────────────┤
│ ステータスバー                      │← 固定 ★v1.5.0
└─────────────────────────────────────┘
```

**レイアウトの特徴**:
- ツールバー: 画面最上部に固定
- 定員状況ヘッダー: ツールバーの下に固定
- 予定グリッド: 縦横スクロール可能
- ステータスバー: 画面最下部に固定 ★v1.5.0

## 5. UI仕様（操作方法）

### 5.1 操作方法のまとめ

**通泊行**:

| セル状態 | クリック | フォーカス | 右クリック | 備考 |
|---------|---------|-----------|-----------|------|
| **空欄** | ○追加 | ✅ | メニュー | 泊りメニュー追加 |
| **○** | 空欄に戻す | ✅ | メニュー | - |
| **◓/◒** | 空欄に戻す | ✅ | メニュー | - |
| **入** | フォーカスのみ | ✅ | メニュー | 期間クリア |
| **退** | フォーカスのみ | ✅ | メニュー | 期間クリア |
| **◎** | **無反応** | ❌ | メニュー | 期間クリア |

**重要な変更点**:
1. **宿泊期間作成**: 空欄を右クリック→「泊り」→退所日クリック
2. **ドラッグ操作廃止**: すべてのドラッグ操作を削除
3. **期間削除**: 右クリック→クリアのみ
4. **期間調整**: 削除→再作成で対応

---

**訪問行**:

| 操作 | 動作 |
|------|------|
| クリック | フォーカス + 回数+1（0→1→2→3→0） |
| 右クリック | フォーカス + メニュー表示 |

### 5.3 右クリックメニュー

#### 5.3.1 通泊行のメニュー

**通常セル（空欄、○、◓、◒）**:
```
┌──────────┐
│ 泊り      │ → 2ステップクリック開始
│ 通い全日  │ → ○
│ 前半通い  │ → ◓
│ 後半通い  │ → ◒
│ クリア    │ → 空白
│ --------- │
│ 備考      │ → 備考パネル表示
└──────────┘
```

**「泊り」メニューの動作**:
```
ユーザーが「泊り」を選択
  ↓
そのセルが「入」になる
  ↓
ステータスバー表示:
┌────────────────────────────────┐
│ ⏱️ 退所日をクリックしてください │
│    （Escでキャンセル）           │
└────────────────────────────────┘
  ↓
セルが点滅アニメーション
  ↓
ユーザーが退所日をクリック
  ↓
期間作成完了（入→◎→退）
  ↓
ステータスバー非表示
```

---

**宿泊期間中のセル（入、◎、退）**:
```
┌──────────┐
│ クリア    │ → 期間全体を削除
│ --------- │
│ 備考      │ → 備考パネル表示
└──────────┘
```

**設計意図**:
- 宿泊期間は「期間全体」として扱う
- 個別の記号変更は不可（混乱を防ぐ）
- 期間調整は「削除→再作成」で行う

---

#### 5.3.2 訪問行のメニュー

```
┌──────────┐
│ 1回       │ → 1
│ 2回       │ → 2
│ 3回       │ → 3
│ 直接入力  │ → テキスト入力ダイアログ
│ クリア    │ → 空白
│ --------- │
│ 備考      │ → 備考パネル表示

### 5.4 ステータスバー表示 ★v1.5.0で変更

#### 5.4.1 2ステップクリック中

**表示タイミング**: 「泊り」メニュー選択直後

**表示内容**:
```
┌────────────────────────────────┐
│ ⏱️ 退所日をクリックしてください │
│    （Escでキャンセル）           │
└────────────────────────────────┘
```

**表示位置**: 画面下部のステータスバー（左側）

**スタイル**:
- 背景色: `#0066cc`（青）
- 文字色: `#ffffff`（白）
- アイコン: ⏱️（砂時計）

**自動消去**: なし（明示的な操作まで表示）

**消去条件**:
- 退所日をクリック
- Escキーを押す
- 別のセルで「泊り」を選択

---

#### 5.4.2 エラーメッセージ

**無効な退所日を選択した場合**:

│    選択してください              │
└────────────────────────────────┘
```

**別の利用者を選択した場合**:
```
┌────────────────────────────────┐
│ ❌ 同じ利用者のセルを            │
│    選択してください              │
└────────────────────────────────┘
```

**訪問行を選択した場合**:
```
┌────────────────────────────────┐
│ ❌ 通泊行のセルを                │
│    選択してください              │
└────────────────────────────────┘
```

**スタイル**:
- 背景色: `#d32f2f`（赤）
- 文字色: `#ffffff`（白）
- アイコン: ❌（バツマーク）

**表示時間**: 3秒間

### 5.6 ドラッグ操作の削除

**削除した機能**:
- ❌ ドラッグによる宿泊期間作成
- ❌ ドラッグによる入所日調整
- ❌ ドラッグによる退所日調整
- ❌ ドラッグによる期間移動
- ❌ ドラッグ中のプレビュー表示
- ❌ ドラッグイベントリスナー

**削除理由**:
- 実装難易度が高い
- UX向上効果が低い
- 2ステップクリック方式で代替可能

## 📝 実装時の注意事項 ★v1.5.0で更新

### 1. ステータスバー表示 ★NEW

**HTML構造**:
```html
<div id="status-bar" class="status-bar" style="display: none;">
  <span class="status-bar-icon"></span>
  <span class="status-bar-message"></span>
</div>
```

**JavaScript実装**:
```javascript
class StatusBar {
  show(message, icon = '⏱️', isError = false) {
    const statusBar = document.getElementById('status-bar');
    const iconEl = statusBar.querySelector('.status-bar-icon');
    const messageEl = statusBar.querySelector('.status-bar-message');
    
    iconEl.textContent = icon;
    messageEl.textContent = message;
    
    if (isError) {
      statusBar.classList.add('error');
      setTimeout(() => this.hide(), 3000);
    } else {
      statusBar.classList.remove('error');
    }
    
    statusBar.style.display = 'flex';
  }
  
  hide() {
    const statusBar = document.getElementById('status-bar');
    statusBar.style.display = 'none';
    statusBar.classList.remove('error');
  }
}
```

**使用例**:
```javascript
// 2ステップクリック中
statusBar.show('退所日をクリックしてください（Escでキャンセル）', '⏱️');

// エラー表示
statusBar.show('退所日は入所日より後の日付を選択してください', '❌', true);

// 非表示
statusBar.hide();
```
