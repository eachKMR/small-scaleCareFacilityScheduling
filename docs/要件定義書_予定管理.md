## 1. 通泊行の記号定義

### 1.1 基本記号

| 記号 | Unicode | 意味 | 定員カウント |
|------|---------|------|-------------|
| **○** | U+25CB | 通い全日（通いのみ） | 通い: 前半1 + 後半1, 泊り: 0 |
| **◎** | U+25CE | 連泊中（通い+泊り） | 通い: 前半1 + 後半1, 泊り: 1 |
| **◓** | U+25D3 | 前半通い | 通い: 前半1 + 後半0, 泊り: 0 |
| **◒** | U+25D2 | 後半通い | 通い: 前半0 + 後半1, 泊り: 0 |
| **入** | - | 入所（宿泊開始） | 通い: 前半1 + 後半1, 泊り: 1 |
| **退** | - | 退所（宿泊終了） | 通い: 前半1 + 後半1, 泊り: 0 |

**重要な変更点**:
- **◎（連泊中）を追加**: 宿泊期間の中間日を表す
- **○は通いのみ**: 泊りにはカウントされない
- **空白（青背景）は廃止**: 中間日は◎で表示

---

### 1.2 記号の詳細説明

#### 1.2.1 通い全日（○）

**意味**:
- 通いサービスのみを全日利用
- 前半（午前）と後半（午後）の両方を利用
- **泊りにはカウントされない**

**定員カウント**:
- 通い定員：前半+1、後半+1
- 泊り定員：+0

**使用頻度**:
- 最も一般的
- 基本的な通いサービス

**入力方法**:
- 1回目クリック → フォーカス
- フォーカス中に2回目クリック → ○追加（空白セルの場合）
- 右クリックメニュー

---

#### 1.2.2 連泊中（◎）

**意味**:
- 宿泊期間の中間日（入と退の間）
- 通いと泊りの両方にカウントされる
- **手動では入力不可**（ドラッグ操作時に自動設定）

**定員カウント**:
- 通い定員：前半+1、後半+1
- 泊り定員：+1

**使用頻度**:
- 宿泊期間が3日以上の場合に発生
- 例: 「入◎◎退」（4日間の宿泊）

**入力方法**:
- **ドラッグ操作でのみ自動設定**
- 手動では入力不可
- クリックすると○に変換される

**視覚的表示**:
- 薄青背景
- 青い罫線なし

---

#### 1.2.3 前半通い（◓）

**意味**:
- 通いサービスを前半（午前）のみ利用
- 後半（午後）は利用しない

**定員カウント**:
- 通い定員：前半+1、後半+0
- 泊り定員：+0

**使用頻度**:
- レアケース
- 必要になれば、その状況が解消するまでは定期的に発生

**時刻定義**:
- 明確な時刻は定義しない
- 「時間が被らないように利用してください」という意図

**入力方法**:
- **右クリックメニューのみ**
- クリックでは入力不可（誤爆防止）

---

#### 1.2.4 後半通い（◒）

**意味**:
- 通いサービスを後半（午後）のみ利用
- 前半（午前）は利用しない

**定員カウント**:
- 通い定員：前半+0、後半+1
- 泊り定員：+0

**使用頻度**:
- レアケース
- 必要になれば、その状況が解消するまでは定期的に発生

**入力方法**:
- **右クリックメニューのみ**
- クリックでは入力不可（誤爆防止）

---

#### 1.2.5 入所（入）

**意味**:
- 宿泊サービスの開始日
- この日から宿泊期間が始まる

**定員カウント**:
- 通い定員：前半+1、後半+1
- 泊り定員：+1

**自動設定**:
- ドラッグ操作で自動設定される
- 手動入力も可能（右クリックメニュー）

**視覚表現**:
- 文字：「入」
- 背景色：薄青（#d0e8f0）
- 左罫線：青（#0066cc）、太さ3px

---

#### 1.2.6 退所（退）

**意味**:
- 宿泊サービスの終了日
- この日で宿泊期間が終わる

**定員カウント**:
- 通い定員：前半+1、後半+1
- **泊り定員：+0**（★重要：退所日は泊りカウントしない）

**理由**:
- 退所日は朝に退所するため、その日の夜は宿泊しない
- 通いサービスとしてのみカウント

**自動設定**:
- ドラッグ操作で自動設定される
- 手動入力も可能（右クリックメニュー）

**視覚表現**:
- 文字：「退」
- 背景色：薄青（#d0e8f0）
- 左罫線：青（#0066cc）、太さ3px

## 2. 訪問回数の入力

### 2.1 基本仕様

#### 2.1.1 入力方法

**クリック連打** ★v1.1で変更:
```
0 → 1 → 2 → 3 → 0 → 1 → ...
```

**変更理由**:
- 旧仕様: 0→1→2→...→9→0（10段階）
- 新仕様: 0→1→2→3→0（4段階）
- **通常利用では3回までで十分**
- UI操作の煩雑さを軽減

**右クリックメニュー**:
- 4回以上の訪問が必要な場合
- 「直接入力」で数値を手入力

---

#### 2.1.2 フォーカスとカウントアップ ★v1.1で追加

**動作仕様**:
```
空欄セルをクリック → フォーカス + 1
1のセルをクリック → フォーカス + 2
2のセルをクリック → フォーカス + 3
3のセルをクリック → フォーカス + 0（クリア）

フォーカス中セルを再クリック → カウントアップ継続
別セルをクリック → フォーカス移動 + そのセルをカウントアップ
```

**視覚的フィードバック**:
- フォーカス中のセルは青枠で強調表示
- どのセルが「カレント」か明確

---

### 2.2 表示形式

#### 2.2.1 数字の表示

**通常の訪問**:
```
┌───┬───┬───┬───┐
│ 1 │ 2 │ 3 │   │
└───┴───┴───┴───┘
```

**4回以上の訪問**:
```
┌───┬───┬───┐
│ 4 │ 5 │10 │
└───┴───┴───┘
```

---

#### 2.2.2 訪問なしの表示

**空白**:
```
┌───┐
│   │
└───┘
```

## 3. 重複の可否

### 3.1 通泊行と訪問行の重複

#### 3.1.1 許可される重複

**訪問 + 通い（○）**:
```
┌────┬───┐
│青砥│ ○ │ ← 通泊行：通い全日
│    │ 1 │ ← 訪問行：訪問1回
└────┴───┘
```

**訪問 + 入所（入）**:
```
┌────┬───┐
│青砥│入  │ ← 通泊行：入所
│    │ 1 │ ← 訪問行：訪問1回
└────┴───┘
```

**訪問 + 退所（退）**:
```
┌────┬───┐
│青砥│退  │ ← 通泊行：退所
│    │ 2 │ ← 訪問行：訪問2回
└────┴───┘
```

**訪問 + 宿泊中（◎）**:
```
┌────┬───┐
│青砥│ ◎ │ ← 通泊行：宿泊中
│    │ 1 │ ← 訪問行：訪問1回
└────┴───┘
```

---

#### 3.1.2 実際の使用例

**例1: 通所と訪問の組み合わせ**
```
┌────┬───┬───┬───┐
│青砥│ ○ │ ○ │ ○ │
│    │ 1 │ 2 │   │
└────┴───┴───┴───┘
5日: 通所 + 訪問1回
6日: 通所 + 訪問2回
7日: 通所のみ
```

**例2: 宿泊中の訪問**
```
┌────┬───┬───┬───┬───┐
│青砥│入  │ ◎ │ ◎ │退  │
│    │   │ 1 │ 2 │ 1 │
└────┴───┴───┴───┴───┘
5日: 入所
6日: 宿泊中 + 訪問1回
7日: 宿泊中 + 訪問2回
8日: 退所 + 訪問1回
```

### 4.2 月をまたぐ宿泊

#### 4.2.1 右特別セルへのドラッグ

**31日から右特別セルへドラッグ**:
```
31日をマウスダウン → 右特別セルへドラッグ
  ↓
カレンダーピッカー表示
```

**カレンダーピッカー**:
```
┌─────────────────┐
│  2026年1月       │
│ 日 月 火 水 木 金土│

  ↓

```

**制限**:
- **ドラッグ開始日の翌日以降**しか選択不可
- 例: 5日まで → 11/6以降選択可能

## 5. 削除⇔復元機能

### 5.1 基本仕様

#### 5.1.1 削除⇔復元の流れ ★v1.1で変更

```
通常状態
  ↓ 1回目クリック
フォーカス状態（青枠表示）
  ↓ フォーカス中に2回目クリック
削除状態（グレー、点滅）
  ↓ 5秒以内にクリック
通常状態（復元）
  ↓ 5秒経過
完全削除（復元不可）
```

---

#### 5.1.2 復元タイムアウト

**設定値**:
- **5秒**（AppConfig.RESTORE_TIMEOUT = 5000）

**調整方法**:
```javascript
const AppConfig = {
  RESTORE_TIMEOUT: 5000,  // ミリ秒（調整可能）
  // ...
};
```

**将来的な拡張**:
- 設定画面で調整可能にする可能性あり
- 現時点では固定値

---

### 5.4 実装の詳細

#### 5.4.1 ScheduleCellのデータ構造

```javascript
class ScheduleCell {
  constructor(userId, date, cellType) {
    this.userId = userId;
    this.date = date;
    this.cellType = cellType;
    this.inputValue = "";
    this.note = "";
    this.actualFlags = {
      day: false,
      stay: false,
      visit: 0,
      halfDayType: 'full'
    };
    
    // 削除⇔復元機能
    this.deletedValue = null;  // 削除前の値
    this.deletedAt = null;     // 削除時刻（ミリ秒）
  }
  
  // 削除
  delete() {
    if (this.inputValue !== "") {
      this.deletedValue = this.inputValue;
      this.deletedAt = Date.now();
      this.inputValue = "";
      Logger?.debug(`Cell deleted: ${this.userId} ${this.date}`);
    }
  }
  
  // 復元
  restore() {
    if (this.canRestore()) {
      this.inputValue = this.deletedValue;
      this.deletedValue = null;
      this.deletedAt = null;
      Logger?.info(`Cell restored: ${this.userId} ${this.date}`);
      return true;
    }
    return false;
  }
  
  // 復元可能判定
  canRestore() {
    if (this.deletedValue === null) return false;
    const elapsed = Date.now() - this.deletedAt;
    return elapsed < AppConfig.RESTORE_TIMEOUT;
  }
  
  // 削除状態のクリア（タイムアウト時）
  clearDeletedState() {
    this.deletedValue = null;
    this.deletedAt = null;
  }
}
```

#### 5.4.2 タイムアウト処理

**実装**:
```javascript
class CellEditor {
  constructor(app) {
    this.app = app;
    this.focusedCell = null;     // v1.1で追加
    this.restoreTimers = new Map();  // userId_date → timerId
  }
  
  handleCellClick(cell) {
    // フォーカス中のセルを再クリック → 操作実行
    if (this.focusedCell === cell) {
      this.executeAction(cell);
      return;
    }
    
    // 新しいセルをクリック → フォーカス
    this.setFocus(cell);
  }
  
  setFocus(cell) {
    // 前のフォーカスを解除
    if (this.focusedCell) {
      this.focusedCell.element.classList.remove('focused');
    }
    
    // 新しいセルにフォーカス
    this.focusedCell = cell;
    cell.element.classList.add('focused');
    cell.element.focus();
  }
  
  executeAction(cell) {
    if (cell.cellType === 'dayStay') {
      // 通泊行の操作
      if (cell.canRestore()) {
        this.restoreCell(cell);
      } else if (cell.inputValue !== "") {
        this.deleteCell(cell);
      } else {
        cell.inputValue = AppConfig.SYMBOLS.FULL_DAY;
        this.app.controllers.schedule.updateCell(cell);
      }
    }
  }
  
  deleteCell(cell) {
    cell.delete();
    this.app.controllers.schedule.updateCell(cell);
    
    // タイムアウトタイマーを設定
    const key = `${cell.userId}_${cell.date}`;
    const timerId = setTimeout(() => {
      cell.clearDeletedState();
      this.app.controllers.schedule.updateCell(cell);
      this.restoreTimers.delete(key);
      Logger?.info(`Cell restore timeout: ${key}`);
    }, AppConfig.RESTORE_TIMEOUT);
    
    this.restoreTimers.set(key, timerId);
  }
  
  restoreCell(cell) {
    cell.restore();
    this.app.controllers.schedule.updateCell(cell);
    
    // タイムアウトタイマーをキャンセル
    const key = `${cell.userId}_${cell.date}`;
    const timerId = this.restoreTimers.get(key);
    if (timerId) {
      clearTimeout(timerId);
      this.restoreTimers.delete(key);
    }
  }
}
```

### 5.5 画面を閉じた場合の動作

**仕様**:
- 画面を閉じる（ブラウザを閉じる、タブを閉じる）
- → 削除状態はクリア（復元不可）

**理由**:
- LocalStorageにdeletedValueを保存しない
- メモリ上のみで管理

**実装**:
```javascript
// LocalStorageに保存する際、削除状態は除外
toJSON() {
  return {
    userId: this.userId,
    date: this.date,
    cellType: this.cellType,
    inputValue: this.inputValue,
    note: this.note,
    actualFlags: this.actualFlags,
    // deletedValue, deletedAt は保存しない
  };
}
```

## 6. 操作方法の詳細

### 6.1 通泊行の操作 ★v1.1で変更

| 操作 | 動作 |
|------|------|
| **ホバー** | ツールチップ表示 |
| **1回目クリック** | セルにフォーカス（青枠表示） |
| **フォーカス中に2回目クリック** | 削除⇔復元、または○追加 |
| **フォーカス中にドラッグ** | 入所→退所の範囲選択 |
| **右クリック** | フォーカス + メニュー表示 |

**フォーカスの動作** ★v1.1で追加:
- 別のセルをクリック → フォーカスが移動
- フォーカス中のセルを再クリック → 操作実行（削除⇔復元、○追加）
- フォーカス状態は青枠で表示

---

### 6.2 訪問行の操作 ★v1.1で変更

| 操作 | 動作 |
|------|------|
| **クリック** | フォーカス + 回数+1（0→1→2→3→0） |
| **右クリック** | フォーカス + メニュー表示（直接入力可能） |

**フォーカスとカウントアップ** ★v1.1で追加:
- 空欄をクリック → フォーカス + 1
- 1をクリック → フォーカス + 2
- 2をクリック → フォーカス + 3
- 3をクリック → フォーカス + 0（クリア）
- フォーカス中セルの再クリック → カウントアップ継続
- 別セルをクリック → フォーカス移動 + そのセルをカウントアップ
