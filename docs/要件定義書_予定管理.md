## 1. 通泊行の記号定義

### 1.1 基本記号

| 記号 | Unicode | 意味 | 定員カウント | クリック操作 |
|------|---------|------|-------------|-------------|
| **○** | U+25CB | 通い全日（通いのみ） | 通い: 前半1 + 後半1, 泊り: 0 | ✅ 可能 |
| **◎** | U+25CE | 連泊中（通い+泊り） | 通い: 前半1 + 後半1, 泊り: 1 | ❌ 不可 |
| **◓** | U+25D3 | 前半通い | 通い: 前半1 + 後半0, 泊り: 0 | ✅ 可能 |
| **◒** | U+25D2 | 後半通い | 通い: 前半0 + 後半1, 泊り: 0 | ✅ 可能 |
| **入** | - | 入所（宿泊開始） | 通い: 前半1 + 後半1, 泊り: 1 | ✅ 可能 |
| **退** | - | 退所（宿泊終了） | 通い: 前半1 + 後半1, 泊り: 0 | ✅ 可能 |

**重要な設計原則**:
- **クリック可能 = 操作可能**: 空欄、○、◓、◒、入、退
- **クリック不可 = 操作不要**: ◎（宿泊期間の中間日）
- **直感的な理解**: クリックできるかどうかで操作可否を判断

---

### 1.2 記号の詳細説明

#### 1.2.1 通い全日（○） ★v1.2.1で更新

**意味**:
- 通いサービスのみを全日利用
- 前半（午前）と後半（午後）の両方を利用
- **泊りにはカウントされない**

**定員カウント**:
- 通い定員：前半+1、後半+1
- 泊り定員：+0

**使用頻度**:
- 最も一般的
- 基本的な通いサービス

**入力方法** ★v1.2.1でシンプル化:
- **空欄をクリック** → フォーカス + ○
- **○をクリック** → フォーカス + 空欄
- **シンプルなトグル**: 空欄 ⇔ ○
- 右クリックメニュー

**訪問行との一貫性** ★v1.2.1:
```
訪問行: 空欄 → 1 → 2 → 3 → 空欄
通泊行: 空欄 ⇔ ○
```
どちらもシンプルなサイクル、複雑な状態遷移なし

---

#### 1.2.2 連泊中（◎）

**意味**:
- 宿泊期間の中間日（入と退の間）
- 通いと泊りの両方にカウントされる
- **手動では入力不可**（ドラッグ操作時に自動設定）
- **クリック操作不可**

**定員カウント**:
- 通い定員：前半+1、後半+1
- 泊り定員：+1

**使用頻度**:
- 宿泊期間が3日以上の場合に発生
- 例: 「入◎◎退」（4日間の宿泊）

**入力・操作方法**:
- **ドラッグ操作でのみ自動設定**
- **クリック不可**: クリックしても無反応
- **フォーカス不可**: フォーカスもできない
- **削除方法**: 宿泊期間全体を右クリック→クリア

**設計意図**:
- ◎は「中間状態」で、ユーザーが直接操作する必要がない
- 入・退のドラッグで間接的に調整される
- クリック不可にすることで、UI設計がシンプルになる
- 誤操作を防止

**視覚的表示**:
- 薄青背景
- カーソル: default（矢印）
- ホバー効果: なし

---

#### 1.2.3 前半通い（◓）

**意味**:
- 通いサービスを前半（午前）のみ利用
- 後半（午後）は利用しない

**定員カウント**:
- 通い定員：前半+1、後半+0
- 泊り定員：+0

**使用頻度**:
- レアケース
- 必要になれば、その状況が解消するまでは定期的に発生

**時刻定義**:
- 明確な時刻は定義しない
- 「時間が被らないように利用してください」という意図

**入力方法**:
- **右クリックメニューのみ**
- クリックでは入力不可（誤爆防止）

---

#### 1.2.4 後半通い（◒）

**意味**:
- 通いサービスを後半（午後）のみ利用
- 前半（午前）は利用しない

**定員カウント**:
- 通い定員：前半+0、後半+1
- 泊り定員：+0

**使用頻度**:
- レアケース
- 必要になれば、その状況が解消するまでは定期的に発生

**入力方法**:
- **右クリックメニューのみ**
- クリックでは入力不可（誤爆防止）

---

#### 1.2.5 入所（入）

**意味**:
- 宿泊サービスの開始日
- この日から宿泊期間が始まる

**定員カウント**:
- 通い定員：前半+1、後半+1
- 泊り定員：+1

**自動設定**:
- 空欄からのドラッグ操作で自動設定される

**手動操作**:
- **入をクリック（1回目）** → フォーカス
- **フォーカス中にドラッグ** → 入所日の調整
  - 左へドラッグ → 入所日を前倒し（期間延長）
  - 右へドラッグ → 入所日を後ろ倒し（期間短縮）
- 右クリックメニュー

**視覚表現**:
- 文字：「入」
- 背景色：薄青（#d0e8f0）
- 左罫線：青（#0066cc）、太さ3px

---

#### 1.2.6 退所（退）

**意味**:
- 宿泊サービスの終了日
- この日で宿泊期間が終わる

**定員カウント**:
- 通い定員：前半+1、後半+1
- **泊り定員：+0**（★重要：退所日は泊りカウントしない）

**理由**:
- 退所日は朝に退所するため、その日の夜は宿泊しない
- 通いサービスとしてのみカウント

**自動設定**:
- 空欄からのドラッグ操作で自動設定される

**手動操作**:
- **退をクリック（1回目）** → フォーカス
- **フォーカス中にドラッグ** → 退所日の調整
  - 左へドラッグ → 退所日を短縮（期間短縮）
  - 右へドラッグ → 退所日を延長（期間延長）
- 右クリックメニュー

**視覚表現**:
- 文字：「退」
- 背景色：薄青（#d0e8f0）
- 右罫線：青（#0066cc）、太さ3px

## 2. 訪問回数の入力

### 2.1 基本仕様

#### 2.1.1 入力方法

**クリック連打**:
```
0 → 1 → 2 → 3 → 0 → 1 → ...
```

**変更理由**:
- 旧仕様: 0→1→2→...→9→0（10段階）
- 新仕様: 0→1→2→3→0（4段階）
- **通常利用では3回までで十分**
- UI操作の煩雑さを軽減

**右クリックメニュー**:
- 4回以上の訪問が必要な場合
- 「直接入力」で数値を手入力

---

#### 2.1.2 フォーカスとカウントアップ

**動作仕様**:
```
空欄セルをクリック → フォーカス + 1
1のセルをクリック → フォーカス + 2
2のセルをクリック → フォーカス + 3
3のセルをクリック → フォーカス + 0（クリア）
```

**通泊行との一貫性** ★v1.2.1:
```
訪問行: クリックで値が変化（0→1→2→3→0）
通泊行: クリックで値が変化（空欄⇔○）

どちらもシンプルで一貫性がある
```

**視覚的フィードバック**:
- フォーカス中のセルは青枠で強調表示
- どのセルが「カレント」か明確

---

### 2.2 表示形式

#### 2.2.1 数字の表示

**通常の訪問**:
```
┌───┬───┬───┬───┐
│ 1 │ 2 │ 3 │   │
└───┴───┴───┴───┘
```

**4回以上の訪問**:
```
┌───┬───┬───┐
│ 4 │ 5 │10 │
└───┴───┴───┘
```

## 3. 重複の可否

### 3.1 通泊行と訪問行の重複

#### 3.1.1 許可される重複

**訪問 + 通い（○）**:
```
┌────┬───┐
│青砥│ ○ │ ← 通泊行：通い全日
│    │ 1 │ ← 訪問行：訪問1回
└────┴───┘
```

**訪問 + 入所（入）**:
```
┌────┬───┐
│青砥│入  │ ← 通泊行：入所
│    │ 1 │ ← 訪問行：訪問1回
└────┴───┘
```

**訪問 + 退所（退）**:
```
┌────┬───┐
│青砥│退  │ ← 通泊行：退所
│    │ 2 │ ← 訪問行：訪問2回
└────┴───┘
```

**訪問 + 宿泊中（◎）**:
```
┌────┬───┐
│青砥│ ◎ │ ← 通泊行：宿泊中
│    │ 1 │ ← 訪問行：訪問1回
└────┴───┘
```

## 4. 宿泊期間の入力方法

### 4.1 新規宿泊期間の作成

#### 4.1.1 空欄からのドラッグ

**基本動作**:
```
空欄をクリック（マウスダウン） → ドラッグ → 別セルでマウスアップ
```

**結果**:
```
Before:
┌───┬───┬───┬───┐
│   │   │   │   │
└───┴───┴───┴───┘

操作: 5日の空欄からドラッグ → 8日で離す

After:
┌───┬───┬───┬───┐
│入  │ ◎ │ ◎ │退  │
└───┴───┴───┴───┘
5日  6日  7日  8日
```

**最短宿泊（1泊2日）**:
```
操作: 1日の空欄からドラッグ → 2日で離す

After:
┌───┬───┐
│入  │退  │
└───┴───┘
1日  2日
```

---

#### 4.1.2 逆方向ドラッグ

**退所から入所へドラッグ**:
```
操作: 8日の空欄からドラッグ → 5日で離す

結果は同じ:
┌───┬───┬───┬───┐
│入  │ ◎ │ ◎ │退  │
└───┴───┴───┴───┘
5日  6日  7日  8日
```

---

### 4.2 既存宿泊期間の調整

#### 4.2.1 入所日の調整

**入をクリックしてドラッグ**:

**パターンA: 入所日を前倒し（期間延長）**
```
Before: [空][空][入][◎][◎][退]
              3日 4日 5日 6日

操作: 3日(入)をクリック → 1日へドラッグ

After: [入][◎][◎][◎][◎][退]
      1日 2日 3日 4日 5日 6日
```

**パターンB: 入所日を後ろ倒し（期間短縮）**
```
Before: [入][◎][◎][退]
        1日 2日 3日 4日

操作: 1日(入)をクリック → 3日へドラッグ

After: [空][空][入][退]
      1日 2日 3日 4日
```

---

#### 4.2.2 退所日の調整

**退をクリックしてドラッグ**:

**パターンA: 退所日を延長（期間延長）**
```
Before: [入][◎][◎][退][空][空]
        1日 2日 3日 4日 5日 6日

操作: 4日(退)をクリック → 6日へドラッグ

After: [入][◎][◎][◎][◎][退]
      1日 2日 3日 4日 5日 6日
```

**パターンB: 退所日を短縮（期間短縮）**
```
Before: [入][◎][◎][◎][退]
        1日 2日 3日 4日 5日

操作: 5日(退)をクリック → 3日へドラッグ

After: [入][◎][退][空][空]
      1日 2日 3日 4日 5日
```

### 4.3 月をまたぐ宿泊

#### 4.3.1 右特別セルへのドラッグ

**31日から右特別セルへドラッグ**:
```
Before: [入][◎][◎][空][特別]
        29  30  31  空   翌月

操作: 31日(退)をクリック → 右特別セルへドラッグ
  ↓
カレンダーピッカー表示
```

**カレンダーピッカー**:
```
┌─────────────────┐
│  2026年1月       │
│ 日 月 火 水 木 金土│
│          1  2  3 │
│  4  5  6  7  8 ... 
│                  │
│ 退所日を選択:    │

**制限**:
- **ドラッグ終了日の前日まで**しか選択不可

**左特別セルから日付へドラッグ**:

  ↓

```

**制限**:
- **ドラッグ開始日の翌日以降**しか選択不可
- 例: 5日まで → 11/6以降選択可能

## 5. 操作方法の詳細

### 5.1 通泊行の操作 ★v1.2.1で更新

| セル状態 | クリック可否 | クリック動作 | ドラッグ | 右クリック |
|---------|------------|------------|---------|-----------|
| **空欄** | ✅ 可能 | フォーカス + ○ | 宿泊期間作成 | メニュー表示 |
| **○** | ✅ 可能 | フォーカス + 空欄 | - | メニュー表示 |
| **◓/◒** | ✅ 可能 | フォーカス + 空欄 | - | メニュー表示 |
| **入** | ✅ 可能 | フォーカス | 入所日調整 | メニュー表示 |
| **退** | ✅ 可能 | フォーカス | 退所日調整 | メニュー表示 |
| **◎** | ❌ 不可 | 無反応 | - | メニュー表示（期間クリア） |

**重要な設計原則** ★v1.2.1:
1. **空欄のクリック**: フォーカス + ○追加
2. **○のクリック**: フォーカス + 空欄に戻す（シンプルなトグル）
3. **空欄からドラッグ**: 宿泊期間作成
4. **入・退のドラッグ**: 期間調整
5. **◎はクリック不可**: 操作不要な中間状態
6. **宿泊期間削除**: 右クリック→クリアのみ

**訪問行との一貫性** ★v1.2.1:
```
訪問行: クリックで値が循環（0→1→2→3→0）
通泊行: クリックで値が切り替わる（空欄⇔○）
```

---

### 5.2 訪問行の操作

| 操作 | 動作 |
|------|------|
| **クリック** | フォーカス + 回数+1（0→1→2→3→0） |
| **右クリック** | フォーカス + メニュー表示（直接入力可能） |

**フォーカスとカウントアップ**:
- 空欄をクリック → フォーカス + 1
- 1をクリック → フォーカス + 2
- 2をクリック → フォーカス + 3
- 3をクリック → フォーカス + 0（クリア）
