## 2. 前半後半の分離カウント

### 2.1 基本概念

#### 2.1.1 前半・後半の定義

**時刻**:
- 明確な時刻は定義しない
- 「時間が被らないように利用してください」という意図

**一般的な理解**:
- 前半：午前（概ね朝〜昼）
- 後半：午後（概ね昼〜夕方）

---

#### 2.1.2 分離カウントの理由

**背景**:
- 定員15人は「同時に15人」ではない
- 前半に15人、後半に別の15人が利用可能
- 最大30人/日の利用が可能（理論値）

**例**:
```
前半（午前）:
  青柳、安藤、天野、...（15人）

後半（午後）:
  井上、上田、江藤、...（別の15人）

→ 合計30人が利用可能
```

---

### 2.2 記号別のカウント方法

| 記号 | 前半カウント | 後半カウント | 合計 |
|------|------------|-------------|------|
| ○（通い全日） | +1 | +1 | 1人分 |
| ◓（前半通い） | +1 | +0 | 0.5人分 |
| ◒（後半通い） | +0 | +1 | 0.5人分 |
| 入（入所） | +1 | +1 | 1人分 |
| 退（退所） | +1 | +1 | 1人分（泊りは+0） |
| ○（宿泊中） | +1 | +1 | 1人分 |

---

### 2.3 前半後半カウントの例

#### 2.3.1 シナリオ1: 全日利用のみ

**データ**:
```
5日:
  青柳 ○（全日）
  安藤 ○（全日）
  天野 ○（全日）
```

**カウント結果**:
- 前半：3人
- 後半：3人
- 最大値：3人

**判定**:
- 定員15人に対して3人
- 利用率：3/15 = 20%
- 記号：◎（余裕）

---

#### 2.3.2 シナリオ2: 前半後半の混在

**データ**:
```
5日:
  青柳 ○（全日）    → 前半+1、後半+1
  安藤 ◓（前半のみ） → 前半+1、後半+0
  天野 ◒（後半のみ） → 前半+0、後半+1
```

**カウント結果**:
- 前半：2人（青柳、安藤）
- 後半：2人（青柳、天野）
- 最大値：2人

**判定**:
- 定員15人に対して2人
- 利用率：2/15 = 13%
- 記号：◎（余裕）

---

#### 2.3.3 シナリオ3: 前半が多い

**データ**:
```
5日:
  青柳 ○（全日）    → 前半+1、後半+1
  安藤 ○（全日）    → 前半+1、後半+1
  天野 ○（全日）    → 前半+1、後半+1
  ...（合計13人が全日）
  井上 ◓（前半のみ） → 前半+1、後半+0
  上田 ◓（前半のみ） → 前半+1、後半+0
```

**カウント結果**:
- 前半：15人（13人全日 + 2人前半のみ）
- 後半：13人（13人全日のみ）
- 最大値：15人

**判定**:
- 定員15人に対して15人
- 利用率：15/15 = 100%
- 記号：×（満員）

---

#### 2.3.4 シナリオ4: 定員オーバー（前半のみ）

**データ**:
```
5日:
  青柳 ○（全日）    → 前半+1、後半+1
  安藤 ○（全日）    → 前半+1、後半+1
  天野 ○（全日）    → 前半+1、後半+1
  ...（合計14人が全日）
  井上 ◓（前半のみ） → 前半+1、後半+0
  上田 ◓（前半のみ） → 前半+1、後半+0
```

**カウント結果**:
- 前半：16人（14人全日 + 2人前半のみ）
- 後半：14人（14人全日のみ）
- 最大値：16人

**判定**:
- 定員15人に対して16人
- 利用率：16/15 = 107%
- 記号：×（満員、オーバー）
- **前半のみオーバー**

### 2.4 定員判定のロジック

#### 2.4.1 判定基準

**前半・後半の最大値で判定**

**理由**:
- 退所日は朝に退所するため、その日の夜は宿泊しない
- 通いサービスとしてのみカウント

---

#### 5.3.2 実装

**ScheduleCalendar.calculateAllFlags()**:
```javascript
calculateAllFlags() {
  // 全セルのフラグをリセット
  for (const cell of this.cells.values()) {
    cell.actualFlags.day = false;
    cell.actualFlags.stay = false;
    cell.actualFlags.visit = 0;
  }
  
  // 宿泊期間に基づいてフラグを設定
  for (const period of this.stayPeriods) {
    const dates = period.getDatesInMonth(this.yearMonth);
    
    for (const date of dates) {
      const cell = this.getCell(date, 'dayStay');
      if (cell) {
        cell.actualFlags.day = true;
        
        // ★重要：退所日は泊りフラグを立てない
        if (date !== period.endDate) {
          cell.actualFlags.stay = true;
        }
      }
    }
  }
  
  // 通いセルのフラグを設定
  for (const cell of this.cells.values()) {
    if (cell.cellType === 'dayStay' && !cell.isSpecialCell()) {
      if (cell.isFullDay() || cell.isMorning() || cell.isAfternoon()) {
        cell.actualFlags.day = true;
      }
    }
  }
  
  // 訪問セルのフラグを設定
  for (const cell of this.cells.values()) {
    if (cell.cellType === 'visit' && !cell.isSpecialCell()) {
      const visitCount = parseInt(cell.inputValue) || 0;
      cell.actualFlags.visit = visitCount;
    }
  }
}
```

### 5.4 月間サマリーの計算

#### 5.4.1 ServiceCapacity.getSummary()

```javascript
getSummary() {
  const yearMonth = this.calendars[0]?.yearMonth;
  if (!yearMonth) return null;
  
  const capacities = this.checkMonth(yearMonth);
  
  return {
    totalDays: capacities.length,
    overCapacityDays: capacities.filter(c => c.isOverCapacity()).length,
    
    // 平均利用者数
    averageDayCount: capacities.reduce((sum, c) => sum + c.dayCount, 0) / capacities.length,
    averageDayCountMorning: capacities.reduce((sum, c) => sum + c.dayCountMorning, 0) / capacities.length,
    averageDayCountAfternoon: capacities.reduce((sum, c) => sum + c.dayCountAfternoon, 0) / capacities.length,
    
    // 最大利用者数
    maxDayCount: Math.max(...capacities.map(c => c.getMaxDayCount())),
    maxStayCount: Math.max(...capacities.map(c => c.stayCount)),
    
    // 定員利用率
    dayUtilizationRate: (capacities.reduce((sum, c) => sum + c.getMaxDayCount(), 0) / (capacities.length * 15)) * 100,
    stayUtilizationRate: (capacities.reduce((sum, c) => sum + c.stayCount, 0) / (capacities.length * 9)) * 100
  };
}
```

class DailyCapacity {
  getMaxDayCount() {
    return Math.max(this.dayCountMorning, this.dayCountAfternoon);
  }
  
  isOverCapacity() {
    const maxDayCount = this.getMaxDayCount();
    return maxDayCount > this.dayLimit || this.stayCount > this.stayLimit;
  }
  
  getDayOverflow() {
    const maxDayCount = this.getMaxDayCount();
    return Math.max(0, maxDayCount - this.dayLimit);
  }
}

getUtilizationRate() {

### 3.5 実装の詳細

#### 3.5.1 DailyCapacityクラス

```javascript
class DailyCapacity {
  constructor(date, dayLimit = 15, stayLimit = 9) {
    this.date = date;
    
    // 前半・後半の分離カウント
    this.dayCountMorning = 0;      // 前半の通い利用者数
    this.dayCountAfternoon = 0;    // 後半の通い利用者数
    this.dayCount = 0;             // 全日の通い利用者数（互換性のため）
    
    this.stayCount = 0;            // 泊り利用者数
    this.visitCount = 0;           // 訪問回数合計
    this.dayLimit = dayLimit;
    this.stayLimit = stayLimit;
  }
  
  // 前半後半の最大値を取得
  getMaxDayCount() {
    return Math.max(this.dayCountMorning, this.dayCountAfternoon);
  }
  
  // 定員オーバー判定
  isOverCapacity() {
    const maxDayCount = this.getMaxDayCount();
    return maxDayCount > this.dayLimit || this.stayCount > this.stayLimit;
  }
  
  // 定員状況の記号を取得
  getCapacitySymbol() {
    const maxDayCount = this.getMaxDayCount();
    const rate = (maxDayCount / this.dayLimit) * 100;
    
    if (rate <= AppConfig.CAPACITY.THRESHOLDS.GOOD) return '◎';
    if (rate <= AppConfig.CAPACITY.THRESHOLDS.OK) return '○';
    if (rate <= AppConfig.CAPACITY.THRESHOLDS.WARN) return '△';
    return '×';
  }
  
  // ツールチップ用の詳細データ
  getTooltipData() {
    return {
      full: this.dayCount,
      morning: this.dayCountMorning,
      afternoon: this.dayCountAfternoon,
      morningOverflow: Math.max(0, this.dayCountMorning - this.dayLimit),
      afternoonOverflow: Math.max(0, this.dayCountAfternoon - this.dayLimit)
    };
  }
}
```

## 5. 定員カウントの計算ロジック

### 5.1 ServiceCapacity.checkDate()

```javascript
class ServiceCapacity {
  constructor(calendars) {
    this.calendars = calendars;  // ScheduleCalendar[]
  }
  
  checkDate(date) {
    const capacity = new DailyCapacity(date);
    
    this.calendars.forEach(calendar => {
      const cell = calendar.getCell(date, 'dayStay');
      
      if (cell && cell.actualFlags.day) {
        // 前半後半の判定
        const contribution = cell.getDayCountContribution();
        capacity.dayCountMorning += contribution.morning;
        capacity.dayCountAfternoon += contribution.afternoon;
        
        // 互換性のため全日カウントも更新
        if (contribution.morning === 1 && contribution.afternoon === 1) {
          capacity.dayCount++;
        } else {
          capacity.dayCount += 0.5;
        }
      }
      
      if (cell && cell.actualFlags.stay) {
        capacity.stayCount++;
      }
      
      const visitCell = calendar.getCell(date, 'visit');
      if (visitCell) {
        capacity.visitCount += visitCell.actualFlags.visit;
      }
    });
    
    return capacity;
  }
}
```

### 5.2 ScheduleCell.getDayCountContribution()

```javascript
class ScheduleCell {
  getDayCountContribution() {
    // 特別セルは当月の定員カウントに含めない
    if (this.isSpecialCell()) {
      return { morning: 0, afternoon: 0 };
    }
    
    if (this.isFullDay() || this.isStayStart()) {
      return { morning: 1, afternoon: 1 };
    } else if (this.isStayEnd()) {
      // ★重要：退所日は通いカウントのみ、泊りカウントなし
      // ただし通いとしては前半+1、後半+1
      return { morning: 1, afternoon: 1 };
    } else if (this.isMorning()) {
      return { morning: 1, afternoon: 0 };
    } else if (this.isAfternoon()) {
      return { morning: 0, afternoon: 1 };
    } else if (this.actualFlags.stay) {
      // 宿泊中（○）
      return { morning: 1, afternoon: 1 };
    }
    
    return { morning: 0, afternoon: 0 };
  }
}
```

### 5.3 退所日の扱い（★重要）

#### 5.3.1 ルール

**退所日は泊りカウントしない**

**理由**:
- 退所日は朝に退所するため、その日の夜は宿泊しない
- 通いサービスとしてのみカウント

**通いカウント**:
- 退所日は通い定員にカウント（前半+1、後半+1）
- 朝〜昼の時間帯は施設を利用するため

**泊りカウント**:
- 退所日は泊り定員にカウント**しない**
- その日の夜は宿泊しないため

---

#### 5.3.2 実装

**ScheduleCalendar.calculateAllFlags()**:
```javascript
calculateAllFlags() {
  // 全セルのフラグをリセット
  for (const cell of this.cells.values()) {
    cell.actualFlags.day = false;
    cell.actualFlags.stay = false;
    cell.actualFlags.visit = 0;
  }
  
  // 宿泊期間に基づいてフラグを設定
  for (const period of this.stayPeriods) {
    const dates = period.getDatesInMonth(this.yearMonth);
    
    for (const date of dates) {
      const cell = this.getCell(date, 'dayStay');
      if (cell) {
        cell.actualFlags.day = true;
        
        // ★重要：退所日は泊りフラグを立てない
        if (date !== period.endDate) {
          cell.actualFlags.stay = true;
        }
      }
    }
  }
  
  // 通いセルのフラグを設定
  for (const cell of this.cells.values()) {
    if (cell.cellType === 'dayStay' && !cell.isSpecialCell()) {
      if (cell.isFullDay() || cell.isMorning() || cell.isAfternoon()) {
        cell.actualFlags.day = true;
      }
    }
  }
  
  // 訪問セルのフラグを設定
  for (const cell of this.cells.values()) {
    if (cell.cellType === 'visit' && !cell.isSpecialCell()) {
      const visitCount = parseInt(cell.inputValue) || 0;
      cell.actualFlags.visit = visitCount;
    }
  }
}
```

### 5.4 月間サマリーの計算

#### 5.4.1 ServiceCapacity.getSummary()

```javascript
getSummary() {
  const yearMonth = this.calendars[0]?.yearMonth;
  if (!yearMonth) return null;
  
  const capacities = this.checkMonth(yearMonth);
  
  return {
    totalDays: capacities.length,
    overCapacityDays: capacities.filter(c => c.isOverCapacity()).length,
    
    // 平均利用者数
    averageDayCount: capacities.reduce((sum, c) => sum + c.dayCount, 0) / capacities.length,
    averageDayCountMorning: capacities.reduce((sum, c) => sum + c.dayCountMorning, 0) / capacities.length,
    averageDayCountAfternoon: capacities.reduce((sum, c) => sum + c.dayCountAfternoon, 0) / capacities.length,
    averageStayCount: capacities.reduce((sum, c) => sum + c.stayCount, 0) / capacities.length,
    
    // 最大利用者数
    maxDayCount: Math.max(...capacities.map(c => c.getMaxDayCount())),
    maxStayCount: Math.max(...capacities.map(c => c.stayCount)),
    
    // 定員利用率
    dayUtilizationRate: (capacities.reduce((sum, c) => sum + c.getMaxDayCount(), 0) / (capacities.length * this.dayLimit)) * 100,
    stayUtilizationRate: (capacities.reduce((sum, c) => sum + c.stayCount, 0) / (capacities.length * this.stayLimit)) * 100,
    
    // 訪問合計
    totalVisits: capacities.reduce((sum, c) => sum + c.visitCount, 0)
  };
}
```
