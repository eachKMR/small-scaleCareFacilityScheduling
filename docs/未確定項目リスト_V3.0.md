## ✅ Phase 0で確定した項目

### データ管理
- ✅ **DEFAULT_USERSの廃止** - 利用者マスタは空で起動
- ✅ **1ヶ月分のみ保持** - 容量超過時は自動削除
- ✅ **セル入力ごとに自動保存** - リアルタイム保存
- ✅ **起動時の画面** - Option A（空のグリッド + メッセージ）採用

### 月間管理
- ✅ **33セル構造** - 左特別 + 1〜31日 + 右特別
- ✅ **特別セルキー** - `prevMonth`, `nextMonth`
- ✅ **月またぎ宿泊** - 特別セルで管理

### 予定入力
- ✅ **記号** - ○◓◒入退（Unicode）
- ✅ **前半後半通い** - 右クリックメニューのみ
- ✅ **削除⇔復元** - 5秒間トグル可能
- ✅ **宿泊期間削除** - 期間単位で削除⇔復元

### 定員管理
- ✅ **前半・後半カウント** - 分離カウント
- ✅ **定員判定** - 最大値で判定
- ✅ **退所日の扱い** - 泊りカウントなし
- ✅ **定員オーバー時** - 入力制限なし

### UI
- ✅ **月選択UI** - Option A（ドロップダウン、前後3ヶ月）採用
- ✅ **グリッドサイズ** - 58行 × 33列
- ✅ **定員状況ヘッダー** - 固定表示（sticky）

### CSV取り込み関連

#### 1. CSVファイル選択後の処理フロー

**現状**: 未確定  
**Phase 1-Aでの対応**: 実装しながら確定

**Option A: 直ちに取り込み（推奨）**
```
ファイル選択 → パース → 保存 → グリッド表示
```

**Option B: プレビュー画面を表示**
```
ファイル選択 → パース → プレビュー → [取り込む]ボタン → 保存
```

**推奨**: Option A（シンプルな操作フロー）

**決定期限**: Phase 1-A実装時

---

#### 2. 複数CSVファイルの取り込み方法

**現状**: 未確定  
**Phase 1-Aでの対応**: 実装しながら確定

**Option A: 複数ファイル同時選択（推奨）**
```html
<input type="file" multiple accept=".csv">
```

**Option B: 1つずつ選択**
```
介護.csvを選択 → 取り込み → 予防.csvを選択 → 統合
```

**推奨**: Option A（ユーザー体験向上）

**決定期限**: Phase 1-A実装時
