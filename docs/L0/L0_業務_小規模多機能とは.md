## 1. 小規模多機能型居宅介護とは

### 1.1 概要

**小規模多機能型居宅介護**とは、中重度の要介護者となっても、**在宅での生活が継続できるように支援する**、小規模な居住系サービスの施設です。

#### キーワード

| 項目 | 内容 |
|------|------|
| **対象者** | 中重度の要介護者 |
| **目的** | 在宅での生活継続を支援 |
| **規模** | 小規模（地域密着型） |
| **中心サービス** | デイサービス（通い） |
| **組み合わせ** | 訪問介護、ショートステイ |

---

### 1.2 制度の背景

#### 創設時期

**2006年4月** - 介護保険制度改正により創設

#### 創設の理由

1. **今後増加が見込まれる利用者層**
   - 認知症高齢者
   - 中重度の要介護者

2. **生活継続の支援**
   - 住み慣れた地域での生活を継続
   - 在宅生活を可能な限り長く維持

3. **新たなサービス体系**
   - 地域密着型サービスとして創設
   - 地域に根ざした小規模施設

---

## 2. サービスの特徴

### 2.1 3つのサービスを1つの事業者で提供

**最大の特徴**は、1つの事業者と契約するだけで、以下の3つのサービスを**組み合わせて**利用できることです。

#### サービス構成

```
┌─────────────────────────────────────┐
│      小規模多機能型居宅介護        │
│                                     │
│  ┌──────┐  ┌──────┐  ┌──────┐  │
│  │ 通い │  │ 訪問 │  │泊まり│  │
│  │(基本)│  │(随時)│  │(随時)│  │
│  └──────┘  └──────┘  └──────┘  │
│       ↑          ↑          ↑      │
│       └──────同じスタッフ──────┘      │
└─────────────────────────────────────┘
```

| サービス | 正式名称 | 内容 | 提供頻度 |
|---------|----------|------|----------|
| **通い** | デイサービス | 施設で日中を過ごす | 基本・定期的 |
| **訪問** | 訪問介護 | 自宅への訪問支援 | 随時・必要に応じて |
| **泊まり** | ショートステイ | 施設での宿泊 | 随時・必要に応じて |

**泊まりサービスの居室管理の重要性については、L0_業務_居室管理の重要性.md を参照してください。**

---

### 2.2 利用者の様態や希望に応じた柔軟な対応

要介護者の状態や希望に応じて、サービスを柔軟に組み合わせることができます。

#### 組み合わせの例

**基本パターン**:
- 週3回の「通い」を利用

**体調不良時**:
- 「通い」+ 自宅での「訪問」を追加

**介護者の都合**:
- 「通い」+ 週末の「泊まり」を組み合わせ

**重要なポイント**: 状況に応じて柔軟にサービスを変更できる

## 3. 従来のサービスとの違い

### 3.1 従来の問題点

小規模多機能型居宅介護が創設される**前**は、以下のような問題がありました。

#### 問題1：サービスごとに別の施設

```
従来のサービス利用
  ├─ 通い（デイサービス） → A事業所
  ├─ 訪問（訪問介護）    → B事業所
  └─ 泊まり（ショートステイ）→ C事業所
```

- **それぞれ別の施設**で受けていた
- 利用者に対応する**スタッフが異なる**

#### 問題2：馴染みの関係が保たれない

- ケアの**連続性**が保たれない
- **馴染みの関係**が築けない
- 調整や連携が**複雑**

#### 問題3：認知症高齢者への影響

**特に認知症高齢者の場合**、以下のような問題が発生：

| 症状 | 影響 |
|------|------|
| **記憶の障害** | 自分のいる場所がわからなくなる |
| **認知機能の障害** | 周囲の環境の変化に対応できない |
| **結果** | 不安や混乱を招き、症状の悪化を引き起こす |

---

### 3.2 小規模多機能型居宅介護の利点

従来の問題を解決するため、以下のような特徴があります。

#### 特徴1：地域に根ざした小規模施設

- **地域密着型**のサービス
- 住み慣れた地域での支援

#### 特徴2：同じスタッフが継続的に対応

```
小規模多機能型居宅介護
  ├─ 通い   ┐
  ├─ 訪問   ├─ 同じスタッフが提供
  └─ 泊まり ┘
```

- **同一のスタッフ**が3つのサービスを提供
- 利用者との**馴染みの関係**が構築される

#### 特徴3：連続性のあるケア

- サービス間での**情報共有が円滑**
- ケアの**一貫性**が保たれる
- 利用者の**状態変化に迅速に対応**

#### 特徴4：安心感

- 環境の変化が少ない
- 知っているスタッフがいる
- **心理的な安定**が得られる

## 4. 対象者

### 4.1 利用できる方

小規模多機能型居宅介護を利用できるのは、以下の条件を満たす方です。

#### 条件

| 項目 | 条件 |
|------|------|
| **要介護認定** | 要介護の認定を受けた方 |
| **居住地** | 事業者と同一の市町村に住んでいる方 |

**注意**: 地域密着型サービスのため、**同一市町村**であることが必須です。

---

### 4.2 要支援の方の場合

要支援1または要支援2の認定を受けた方は、**介護予防小規模多機能型居宅介護**のサービスが利用できます。

#### 違い

| 項目 | 要介護 | 要支援 |
|------|--------|--------|
| **サービス名** | 小規模多機能型居宅介護 | **介護予防**小規模多機能型居宅介護 |
| **対象** | 要介護1〜5 | 要支援1〜2 |
| **目的** | 介護サービス | 介護予防 |

## 5. 定員の規定

### 5.1 基本的な定員

小規模多機能型居宅介護には、以下の定員が定められています。

| 定員の種類 | 人数 | 備考 |
|-----------|------|------|
| **登録定員** | 29人以下 | 事業所に登録できる利用者の上限 |
| **通いの利用定員** | 概ね15人以下 | 1日あたりの通いサービス利用者 |
| **泊まりの利用定員** | 概ね9人以下 | 1日あたりの泊まりサービス利用者 |

#### 重要なポイント

- **登録定員**：この事業所を利用できる人の総数
- **通い・泊まり定員**：**1日あたり**の同時利用者数

**詳細な法的枠組みについては、L0_業務_定員の法的枠組み.md を参照してください。**

---

### 5.2 特例（定員の緩和）

一定の条件を満たす場合、通いの定員を緩和できます。

#### 条件

1. **登録定員が26名以上29名以下**
2. **居間および食堂の面積**が定められている広さを確保

#### 緩和内容

- 通いサービスの利用定員を**18人以下**に引き上げ可能
- 標準の15人から**3人増**

---

## 6. 併用可能なサービス

### 6.1 併用不可のサービス

小規模多機能型居宅介護を利用する場合、以下のサービスは**併用できません**。

| サービス | 理由 |
|---------|------|
| **居宅介護支援** | 小規模多機能のケアマネージャーが行う |
| **訪問介護** | 小規模多機能の訪問サービスで提供 |
| **訪問入浴介護** | 小規模多機能の通いサービスで提供 |
| **通所介護（デイサービス）** | 小規模多機能の通いサービスで提供 |
| **通所リハビリテーション（デイケア）** | 小規模多機能の通いサービスで提供 |
| **短期入所生活介護（ショートステイ）** | 小規模多機能の泊まりサービスで提供 |
| **短期入所療養介護** | 小規模多機能の泊まりサービスで提供 |

**重要**: 既存のサービスを利用している場合、小規模多機能に移行する際は、これらのサービスを解約する必要があります。

---

### 6.2 併用可能なサービス

一方で、以下のサービスは小規模多機能型居宅介護と**併用できます**。

#### 併用可能なサービス一覧

| サービス | 内容 |
|---------|------|
| **訪問看護** | 看護師による医療的ケア |
| **訪問リハビリテーション** | リハビリ専門職による機能訓練 |
| **居宅療養管理指導** | 医師・薬剤師等による療養上の指導 |
| **福祉用具貸与** | 車椅子・介護ベッド等のレンタル |

#### 注意点

- 併用の可否について詳しくは、ケアマネージャーに確認が必要
- 医療的ケアが必要な場合、訪問看護との併用が重要

## 7. 費用（自己負担額）

### 7.1 負担額の基本

利用者は、介護保険のルールに従い、**自己負担額**を支払います。

#### 負担割合

- 通常：**1割負担**（所得に応じて2割・3割の場合あり）

---

### 7.2 同一建物居住者の特例

**2015年4月より**、以下のルールが適用されています。

#### ルール

事業者と**同一建物に居住する者**は、同一建物居住者以外の居住する者に比べて**自己負担額が安くなります**。

#### 理由

- 訪問サービスの提供コストが低い
- 移動距離が短い

## 8. まとめ

### 8.1 小規模多機能型居宅介護の本質

小規模多機能型居宅介護は、以下の特徴を持つサービスです。

```
┌──────────────────────────────────┐
│ 小規模多機能型居宅介護の特徴      │
├──────────────────────────────────┤
│ ✓ 中重度の要介護者の在宅生活を支援 │
│ ✓ 通い・訪問・泊まりを一体的に提供 │
│ ✓ 同じスタッフが継続的にケア      │
│ ✓ 地域密着型の小規模施設          │
│ ✓ 柔軟なサービスの組み合わせ      │
└──────────────────────────────────┘
```

---

### 8.2 最大の特徴

**同じスタッフが継続的にケアを提供する**ことで、特に**認知症高齢者**にとって：

- 環境の変化によるストレスを軽減
- 馴染みの関係による安心感
- ケアの質の向上

これらを実現している点が、小規模多機能型居宅介護の最大の特徴です。
